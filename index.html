Tech Stack

HTML (HyperText Markup Language)

HTML merupakan singkatan dari HyperText Markup Language. Ini adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML memberi struktur kepada konten web, menandai elemen-elemen seperti judul, paragraf, daftar, tautan, gambar, formulir, dan banyak lagi. Dengan HTML, pengembang web dapat membuat halaman-halaman web yang dapat diakses melalui internet.

CSS (Cascading Style Sheets)

CSS adalah singkatan dari Cascading Style Sheets. CSS digunakan untuk mengendalikan tata letak, warna, dan gaya elemen-elemen HTML di halaman web. Dengan CSS, pengembang web dapat memberi desain visual yang menarik pada halaman web, mengatur posisi elemen, mengubah warna dan font, menyesuaikan responsivitas, dan memberi efek animasi, sehingga menciptakan pengalaman pengguna yang lebih baik.

JavaScript

JavaScript adalah bahasa pemrograman yang berjalan di sisi klien (client-side) dan dijalankan oleh browser pengguna. Ini memungkinkan pengembang web untuk membuat halaman web interaktif dan dinamis. Dengan JavaScript, pengembang dapat menanggapi tindakan pengguna, memvalidasi formulir, mengubah konten halaman secara dinamis, mengambil data dari server tanpa perlu memuat ulang halaman, serta membuat efek animasi dan permainan web.
